to load assets from TranxPurp:
1. Load Tranx
2. Load Tranx.dateTime.fmtString(fmt) -> some string of the datetime
3. Load transaction purpose icon -> call Tranx.purpose.resourceMapper()
4. Load transaction purpose color
    -> Tranx.purpose.colourInt()
    -> should just outline the actual purpose, not the entire card
5. Load transaction amount ->
    FOR NOW: display as single value:
        Tranx.amount.value.toString() + "." + Tranx.amount.fraction.toString()
        + Tranx.amount.currency
    If you want to display bills/coins:
        Tranx.amount.resourceMapper() -> returns set of images corresponding to that amount visually
6. Tranx.direction.resourceMapper()
    -> gives you the resource for the direction of the transaction
7.  when(Tranx.direction) {
    is direction.INCOMING -> outline whole card with 0xFF4CAF50
    is direction.OUTGOING -> outline whole card with 0xFFF44336
}
